# UUIDBridge ドキュメント

UUIDBridge は、Minecraft サーバー管理者が online-mode と offline-mode の
UUID を移行するときに、プレイヤーに紐づくデータを失わないようにするための
サーバー側ツールです。

対象は Minecraft 1.21.1 で、Fabric、Forge、NeoForge のサーバー jar を提供します。
